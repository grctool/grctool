## Current-State Summary

### Observed Product Behavior

GRCTool is a Go CLI tool that automates compliance evidence collection for SOC2 and ISO27001 frameworks. It syncs data from Tugboat Logic, integrates with GitHub/Terraform/Google Workspace, generates AI-assisted evidence, and submits evidence via custom collector URLs.

### Observed Architecture and Runtime Shape

Hexagonal architecture: CLI layer (Cobra) -> Services/Orchestrator -> Domain Models -> Adapters (Tugboat API, GitHub API, Terraform HCL parser, Google Workspace API). Authentication via browser-based Safari cookie extraction. Local JSON file storage. VCR-based HTTP test mocking.

### Operational and Delivery Context

CLI tool distributed via GoReleaser multi-platform builds. CI/CD via GitHub Actions. Pre-commit hooks via lefthook. Security scanning via gosec, govulncheck, golangci-lint. Manual release process via git tags.
